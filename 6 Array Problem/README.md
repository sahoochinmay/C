# C - programming

## Array Problem 

- P1 - Accept 10 no's in array and calculate their sum.
- P2 - Accept  a 3X3 matrix and display that.
- P3 - Accept  a 2X2 matrix and transpose that.
- P4 - Accept  a 3X3 matrix and find out diagonal sum of matrix.
- P5 - Accept  a 3X3 matrix and print it's upper triangular and lower triangular matrix
- P6 - Accept two  2x2 MATRIX and multiply them.
- P7 - Accept two  3x3 MATRIX and find out their equality.
- P8 - Accept two  3x3 MATRIX and print them using multidimentional array.
